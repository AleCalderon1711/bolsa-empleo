# Bolsa de Empleo

Full-stack job board application built with React, Spring Boot, JWT and MySQL.

## Description

Web platform that connects job seekers with companies. Users can register as a company or job seeker, manage job listings, apply to positions, and track applications. Administrators manage platform access and approvals.

## Features

- Role-based access control (Admin, Company, Job Seeker)
- JWT authentication
- Company dashboard to publish and manage job listings
- Job seeker dashboard with CV, skills and applications
- Admin panel to approve companies and job seekers
- Protected routes on the frontend

## Tech Stack

**Frontend:** React, Vite, JavaScript, React Router, Axios  
**Backend:** Java, Spring Boot, Spring Security, JWT, JPA, REST API  
**Database:** MySQL
